# 🧪 DSA Labs — Applied Data Structures & Algorithms

A personal marathon of DSA-inspired projects. Not just solved problems — real, working applications built on top of data structures and algorithms.

Every project here starts with a concept and ends with something that actually runs.

## 💡 Philosophy

Most DSA practice is isolated — you implement a linked list, tick a box, move on. This repo is different. Every data structure should eventually power something real. A stack becomes an expression evaluator. A graph becomes a pathfinder. A tree becomes a search engine index.

The goal isn't to collect solutions. It's to build fluency.
